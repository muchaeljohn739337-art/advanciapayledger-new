# Ancient Future Bridge Component

## Overview
Interactive visualization showcasing the integration of ancient healthcare wisdom with cutting-edge technology.

## Features
- 6 healthcare concepts bridging traditional and modern approaches
- Interactive concept selection with smooth animations
- Responsive design (mobile & desktop)
- Beautiful gradient UI with Tailwind CSS
- Lucide React icons

## Concepts Covered
1. **Meridian Mapping × Neural Networks** - TCM meets AI diagnostics
2. **Ancient Remedies × Immutable Ledgers** - Herbal medicine with blockchain
3. **Zen Practice × Brain-Computer Interfaces** - Meditation with neurofeedback
4. **Needle Therapy × Nano-Robotics** - Acupuncture precision medicine
5. **Sacred Geometry × IoT Architecture** - Healing temples as smart facilities
6. **Body Constitution × Genetic Analysis** - Ayurvedic doshas with genomics

## Usage

### Direct Page Access
Visit: `/ancient-future`

### Component Import
```tsx
import AncientFutureBridge from '@/components/AncientFutureBridge';

export default function MyPage() {
  return <AncientFutureBridge />;
}
```
